## 关于组件
### 组件的复用
每个组件可以多次复用，并且互不影响
### `data`必须是函数
一个组件的`data`选项必须是函数，因此每个实例可以维护一份被返回对象的独立的拷贝。
### 组件的组织
全局注册与局部注册
全局注册：
```javascript
Vue.component('my-component-name', {
  // ... options ...
})
```
### `prop`向自组件传递数据
1. Prop 是你可以在组件上注册的一些自定义特性。当一个值传递给一个 prop 特性的时候，它就变成了那个组件实例的一个属性。
2. 一个组件默认可以拥有任意数量的 prop，任何值都可以传递给任何 prop。

### 单个根元素
每个组件必须只有一个根元素
